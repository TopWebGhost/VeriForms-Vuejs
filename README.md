## 安装

```bash
git clone https://github.com/hilongjw/vue-material-blog.git
```

通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```

启动服务(http://localhost:3000)
``` bash
npm run dev
```

发布代码
``` bash
npm run build
```
